OSM connection test

This is a diagnostic test tool. It checks whether this application can connect to Online Scout Manager and read information. It will not create, change or delete any information in OSM. Your OSM password is entered on the OSM website and is never supplied to this application.

What will be tested?

  • Secure authentication through the OSM sign-in page.
  • That an access token can be obtained, stored securely and refreshed.
  • The groups and sections your OSM account can reach.
  • The permissions OSM returns for each section, and whether they can be interpreted.
  • One low-risk read only API request.
  • The response format, rate limit information and any deprecation or blocked-client warnings.

Every stage reports Passed, Passed with warning, Failed, Skipped or Cancelled, with a plain English explanation, a permanent message code and a correlation identifier.

What this tool will not do

  • It never asks for your OSM password and has no forgotten-password function — password resets are handled in OSM.
  • It cannot run a write operation. Release 1 has no write endpoint at all.
  • It does not retain member records, and diagnostics are sanitised before they are stored or exported.
