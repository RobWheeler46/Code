Help and message catalogue

Every outcome in this application has a permanent message code. Quote the code and the correlation identifier when you ask for support.

How to read a message

Status
Success, Information, Warning, Error or Critical.
What happened
A plain English explanation of the outcome.
What this means
The likely effect on the current test.
What has not happened
Confirmation that no OSM information was changed.
What you can do
Specific next actions.
Retrying
Whether retrying is safe now, safe after a delay, safe only after configuration changes, not recommended, or disabled.
Message code
A permanent reference such as OSM-AUTH-004.
Correlation identifier
A unique reference for the request, such as OSM-20260723-8F4K2.
